# Image Assets

Place your images here:

## Required Images

1. **epk-cover.jpg**
   - Hero section portrait
   - Recommended size: 800x1200px (3:4 aspect ratio)
   - Format: JPG, optimized for web
   - Used in: Hero section

2. **hero-1.jpg**
   - Featured tiles background
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Format: JPG, optimized for web
   - Used in: Featured tiles section

3. **og-image.jpg**
   - OpenGraph/Social media preview
   - Required size: 1200x630px
   - Format: JPG or PNG
   - Used in: Social media shares, link previews

## Image Optimization Tips

- Use tools like [Squoosh](https://squoosh.app/) or [ImageOptim](https://imageoptim.com/) to compress images
- Keep file sizes under 500KB for better performance
- Use WebP format if possible (Next.js will convert automatically)
- Ensure images are properly cropped and centered

## Placeholder Images

If you don't have images yet, you can:
1. Use a placeholder service like [placeholder.com](https://placeholder.com)
2. Create simple colored gradients as placeholders
3. Use AI image generators for temporary assets

The site will display a gradient fallback if images are missing.
