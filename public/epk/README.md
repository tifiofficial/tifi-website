# EPK PDF

Place your Electronic Press Kit PDF here as `tifi-epk.pdf`

The file should be named exactly: **tifi-epk.pdf**

## EPK Contents Should Include:
- Artist bio
- High-resolution photos
- Music catalog
- Performance history
- Press quotes/reviews
- Contact information
- Social media links
- Technical requirements (for venues)

## File Size
Keep the PDF under 10MB for optimal download speed.
